## 对主题版本的要求
要使用搜索功能对你的主题版本可能有最低要求，我采用的是Next主题，最近更新的，为防止后面出现各种某明奇妙的问题，最高把主题版本更新一下。

## Local Serarch
安装 hexo-generator-searchdb ，在站点的根目录下执行以下命令：
    $ npm install hexo-generator-searchdb --save
编辑 站点配置文件，新增以下内容到任意位置：
    search:
      path: search.xml
      field: post
      format: html
      limit: 10000
编辑 主题配置文件，启用本地搜索功能：
    # Local search
    local_search:
      enable: true
